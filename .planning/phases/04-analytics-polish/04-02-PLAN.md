    4. app/pre-vr/page.tsx -- Add `id="main-content"` to the outer div AND add focus management:
       - Add `id="main-content"` to the `<div className="flex min-h-screen flex-col">` element
       - Add a useEffect that moves focus to the screen heading after each transition:
         ```typescript
         useEffect(() => {
           requestAnimationFrame(() => {
             const heading = document.querySelector('[data-screen-heading]')
             if (heading instanceof HTMLElement) {
               heading.setAttribute('tabindex', '-1')
               heading.focus({ preventScroll: false })
             }
           })
         }, [currentScreen])
         ```
       - This runs after the key={currentScreen} remount completes

    5. app/post-vr/page.tsx -- Add `id="main-content"` to the `<main>` element.

    6. Each screen component heading (ScreenOne through ScreenSix, and PostVR h1) -- Add `data-screen-heading` attribute to the primary h1 or h2 so the focus management useEffect can find it:
       - Read each screen component to find its heading element
       - Add `data-screen-heading` to the first heading in each screen
       - Do NOT add tabindex statically -- the useEffect adds tabindex="-1" dynamically

    7. app/pre-vr/components/ScreenThree.tsx -- Wrap employer card in FocusTrap:
       - Import FocusTrap from 'focus-trap-react'
       - Store a ref to the pin button that was clicked so focus can return to it:
         - Add `pinRefs` useRef<Map<string, HTMLElement>> initialized to `new Map()`
         - In the marker creation loop, store each pin element: `pinRefs.current.set(employer.id, el)`
       - Replace the employer card JSX (the `{employer && (...)}` block) with:
         ```typescript
         {employer && (
           <FocusTrap
             active={true}
             focusTrapOptions={{
               onDeactivate: closeCard,
               escapeDeactivates: true,
               clickOutsideDeactivates: true,
               returnFocusOnDeactivate: true,
               setReturnFocus: () => pinRefs.current.get(selectedEmployer!) || document.body,
             }}
           >
             <div>
               {/* Backdrop overlay */}
               <div className="fixed inset-0 z-40 bg-black/20" onClick={closeCard} aria-hidden="true" />
               {/* Card - add aria-modal="true" */}
               <div role="dialog" aria-labelledby="employer-card-name" aria-modal="true" className="fixed bottom-0 ...">
                 {/* existing card content unchanged */}
               </div>
             </div>
           </FocusTrap>
         )}
         ```
       - Remove the separate Escape key useEffect since FocusTrap handles Escape via escapeDeactivates
       - Add `aria-modal="true"` to the dialog div
