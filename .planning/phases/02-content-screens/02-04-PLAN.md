**Imports:**

```typescript
'use client'

import { useState } from 'react'
import { content } from '@/content/config'
```

**State:**
```typescript
const [checkedItems, setCheckedItems] = useState<string[]>([])
const data = content.postVr
```

**Toggle logic:**
```typescript
const toggleItem = (id: string) => {
  setCheckedItems(prev =>
    prev.includes(id)
      ? prev.filter(i => i !== id)
      : [...prev, id]
  )
}
```

**Layout:**
- Single column, max 640px centered: `max-w-[640px] mx-auto`
- Full page: `min-h-screen flex flex-col px-4 py-8`

**Congrats section:**
- Heading: `text-[28px] md:text-[40px]`, weight 800, line-height 1.2, Navy, centered
- Subtext: `text-[20px] md:text-[24px]`, weight 300, Neutral-5, centered, `mt-4`

**Checklist heading:**
- `text-[20px] md:text-[24px]`, weight 800, Navy, `mt-8`

**Progress count:**
- `text-[16px]`, weight 300, `mt-2`
- When 0 checked: "0 of 6 complete" -- all text in Neutral-4
- When N > 0: The number N uses `text-[var(--myb-primary-blue)]`, rest in Neutral-4
- Implementation:
  ```html
  <p className="text-[16px] font-[300] text-[var(--myb-neutral-4)]">
    <span className={checkedItems.length > 0 ? 'text-[var(--myb-primary-blue)] font-[800]' : ''}>
      {checkedItems.length}
    </span>
    {' '}of {data.checklist.length} complete
  </p>
  ```

**Checklist items:**
- Container: `mt-4 flex flex-col` with 12px gap (`gap-3`)
- Each item is a `<button>` with `role="checkbox"` and `aria-checked`:
  ```html
  <button
    onClick={() => toggleItem(item.id)}
    role="checkbox"
    aria-checked={isChecked}
    className="flex w-full items-center gap-3 border-b border-[var(--myb-neutral-1)] pb-3 text-left
               focus:outline-none focus:ring-[3px] focus:ring-[var(--myb-primary-blue)] rounded"
  >
    {/* Checkbox visual */}
    <div className={cn(
      'flex h-6 w-6 flex-shrink-0 items-center justify-center rounded',
      isChecked
        ? 'bg-[var(--myb-primary-blue)] animate-check-bounce'
        : 'border-2 border-[var(--myb-neutral-2)] bg-white'
    )}>
      {isChecked && (
        <svg width="14" height="14" viewBox="0 0 14 14" fill="none" aria-hidden="true">
          <path d="M2 7L5.5 10.5L12 3.5" stroke="white" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round"/>
        </svg>
      )}
    </div>

    {/* Label */}
    <span className={cn(
      'text-[20px] md:text-[24px] font-[300]',
      isChecked
        ? 'text-[var(--myb-neutral-3)] line-through'
        : 'text-[var(--myb-navy)]'
    )}>
      {item.label}
    </span>
  </button>
  ```

**Checkbox animation (Claude's discretion):** Using the `animate-check-bounce` class from globals.css on the checkbox when checked. The class applies a 250ms scale bounce (1 -> 1.15 -> 1). When unchecked, no animation -- just transition back to empty state.
