**Map implementation approach:**

If mapcn components are available, use them:
```typescript
<Map center={REGINA_CENTER} zoom={ZOOM_LEVEL} interactive={false}
  styles={{ light: 'https://basemaps.cartocdn.com/gl/positron-gl-style/style.json' }}>
  {data.employers.map(employer => (
    <MapMarker key={employer.id} longitude={employer.pinPosition.lng} latitude={employer.pinPosition.lat}>
      <MarkerContent>
        <button onClick={() => setSelectedEmployer(employer.id)} ...>
          pin icon
        </button>
      </MarkerContent>
    </MapMarker>
  ))}
</Map>
```

If mapcn is NOT available (shadcn CLI fails), use maplibre-gl directly:
```typescript
import maplibregl from 'maplibre-gl'
// Use useRef for map container, useEffect to initialize map instance

// Disable interactions: map.scrollZoom.disable(), map.dragPan.disable(), etc.
```

**Pin markers:**
- 40px diameter circle button
- `bg-[var(--myb-primary-blue)]` fill, white icon (use a simple inline SVG pin/map-pin icon)
- `hover:scale-110 transition-transform`
- `active:scale-95`
- `aria-label="View {employer.name}"`
- Min touch target: 44px (pad to 44px if needed)

**Disable all map interactions** (per MAP-04):
If using mapcn: try `interactive={false}` prop first. If that doesn't work, set individual props: `scrollZoom={false} dragPan={false} doubleClickZoom={false} touchZoomRotate={false} keyboard={false}`
If using maplibre-gl directly: after map init, call `map.scrollZoom.disable()`, `map.dragPan.disable()`, `map.doubleClickZoom.disable()`, `map.touchZoomRotate.disable()`, `map.keyboard.disable()`

**Escape key handling:**
```typescript
useEffect(() => {
  if (!selectedEmployer) return
  const handler = (e: KeyboardEvent) => {
    if (e.key === 'Escape') setSelectedEmployer(null)
  }
  document.addEventListener('keydown', handler)
  return () => document.removeEventListener('keydown', handler)
}, [selectedEmployer])
```
