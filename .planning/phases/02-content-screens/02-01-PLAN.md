- The column container scrolls via CSS `transform: translateY(-${digit * 10}%)` with `transition-duration: 2000ms` and `ease-out` easing
- Each digit has a stagger delay: `transitionDelay: ${index * 50}ms`
- Wrap the counter in a container with `overflow: hidden` and `h-[1em]`
- Dollar sign `$` rendered as a static character before the digit columns
- Commas rendered as static characters between appropriate digit groups
- Format the salary amount (72000) as "72,000" -- split into individual characters, render each non-comma char as an OdometerDigit

**Animation trigger:**
- Use `useState(false)` for `isVisible` and `useEffect` to set it to `true` after mount (via `requestAnimationFrame` or a short `setTimeout(0)`)
- When `isVisible` is false, all digits show 0 (translateY at 0%)
- When `isVisible` is true, digits scroll to their target positions

**Stat Badge Cards:**
- Navy (#22224C) background, white text

- Small label below: 14px, weight 300 (also white, slightly lower opacity or just white)

- Cards initially have `opacity: 0` and gain visibility via the `animation-fill-mode: both` (already set by the `both` keyword in the utility class)

**prefers-reduced-motion handling:**
